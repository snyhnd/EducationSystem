EducationSystem は、学年ごとに講座があり、受講したら次に進めていける教育支援システムです。
Laravel を用いたチーム開発で構築しており、feature/佐藤 ブランチでは主に「時間割画面」と「授業画面」の実装を担当しました。UI の調整やレビュー対応も行い、管理者が使いやすい画面設計を意識して開発を進めました。

担当機能（feature/佐藤）
時間割画面の作成
授業情報の表示・編集機能
Blade を用いた UI 調整
レビュー指摘に基づく修正対応

使用技術
PHP 8.x / Laravel 10
MySQL / Eloquent ORM
Blade / Bootstrap
Git / GitHub（ブランチ運用によるチーム開発）

工夫した点
授業と時間割のリレーションを整理し、データ構造を分かりやすく設計
バリデーションやエラーハンドリングを適切に実装し、入力ミスに強い画面を構築
UI/UX を意識し、見やすく操作しやすい画面レイアウトを作成
レビュー指摘に柔軟に対応し、コード品質と可読性を向上
